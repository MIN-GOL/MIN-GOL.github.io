二、Ai语音

# 接口地址

http://api.weijieyue.cn/api/yuying/api.php?

# 接口示例

http://api.weijieyue.cn/api/yuying/api.php?msg=[文字]

# 接口参数

参数 必要性 描述
msg 必要 生成文字语音
pit 不 音调
vol 不 音量
apd 不 语速
per 不 发言人

正确
